# obsidian-things

An Obsidian theme inspired by the beautifully-designed app, Things. 

- obsidian-things (GITHUB) -- https://github.com/colineckert/obsidian-things


--------------------------------------------------------------------------------
**obsidian-things markdown**

**Basic**

- [ ] `[ ]` to-do
- [_] `[_]` to-do (modified)
- [/] `[/]` incomplete
- [x] `[x]` done
- [-] `[-]` canceled
- [>] `[>]` forwarded
- [<] `[<]` scheduling

**Extras**

- [?] `[?]` question
- [!] `[!]` important
- [*] `[*]` star
- ["] `["]` quote
- [l] `[l]` location
- [b] `[b]` bookmark
- [i] `[i]` information
- [S] `[S]` savings
- [I] `[I]` idea
- [p] `[p]` pros
- [c] `[c]` cons
- [f] `[f]` fire
- [k] `[k]` key
- [w] `[w]` win
- [u] `[u]` up
- [d] `[d]` down
- [D] `[D]` draft pull request
- [P] `[P]` open pull request
- [M] `[M]` merged pull request
