====== PlayGround ======

## Task checkbox examples

### Square brackets `[ ]`

# - [x] Done task (❎)
# - [ ] Pending task (🔲)
# - [_] Pending task alt (🔲)
# - [>] In progress (🟧)
# - [-] Cancelled (❌)

### Round brackets `( )`

# - (x) Done task (🟢)
# - (>) In progress (🟠)
# - (-) Blocked (🔴)
# - (_) Pending (⭕)
